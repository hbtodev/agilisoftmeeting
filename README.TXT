Site contendo informação a respeito de licença e aquisição da versão PRO.
https://jitsi.org/api/

De acordo com a descrição do site o acesso a api é free, apenas deve-se manter o logo que aparece durante a meeting
e a pagina final que aparece no final da meeting, caso não deseje deve-se adiquirir a versão PRO.

Atualmente o dominio utilizad é 'meet.jit.si', é possivel hospedar a api no dominio proprio no entanto para isso 
deve-se adquirir a versão PRO.

Todas as informações estão mais detalhadas no site informado acima.
